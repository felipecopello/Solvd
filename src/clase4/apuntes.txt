ABSTRACT CLASSES
entities for abstract clases, tienen metodos abstractos tambien. las usamos para evitar duplicar el codigo
para crear una clase abstracta:

public abstract class Car{
	private string color; // tiene campos como cualquier otra clase 
	public Car (string color){
		this.color = color; // tambien tiene un constructor
	} 

	// y setters y getters 
}

NO pueden instanciar objetos, las usamos de superclase para que hereden sus hijas info y metodos.
